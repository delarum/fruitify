# fruitify
This simple fruits website to demo Tailwind CSS capabilities.

## Pages
1. Home Page 
    - Hero section
    - Featured fruit cards
    - testimonials
    - call to action buttons(CTA)

2. Shop/Catalogue(products page)
    - Multi-column layout
    - sidebar filters
    - sale badges - add to card 

3. Product detail 
    - large galleries
    - quantity selectors
    - related items

4. Cart/checkout
    - Order summary tables
    - input fields
    - Sticky checkout buttons

5. Responsive Navbar and Footer
